# PaginaPersonal_V2

Pagina web propia.

## Instalación y Uso
1. Clona el repositorio:
    ```
    git clone https://github.com/ignacioVillarreal2003/myPage2.git
    ```

2. Navega al directorio del proyecto:
    ```
    cd PaginaPersonal_V2
    ```

3. Instala las dependencias:
    ```
    npm install
    ```

4. Para abrir la aplicación:
    ```
    ng serve --open
    ```
